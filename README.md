# Inventory Control Management SQL Project

A simple SQL-based inventory control management system designed to demonstrate database modeling, schema creation, sample data insertion, and essential inventory queries.

This project includes:
✔ Database table creation  
✔ Sample data population  
✔ SQL queries to manage and analyze inventory  

---

##  Project Structure
Inventory-control-management-sql/
├── create_tables.sql — Defines database schema
├── insert_data.sql — Inserts sample data into tables
├── queries.sql — Useful inventory queries
├── .gitignore
└── README.md

---

##  Database Schema

The system includes the following core tables:

| Table Name         | Description |
|--------------------|-------------|
| `products`         | List of products with IDs and names |
| `suppliers`        | Suppliers providing the products |
| `inventory`        | Current product stock levels |
| `transactions`     | Records of stock changes (in/out) |
